Utili::LogCmdtOO

usage: initialize, open, write from main

        our $log = Utili::LogCmdtOO->new(1, "../log/log.csv", 1, 0);
        $log->logOpen();
        $log->logWrite($config{tool}, "start..."); #=item nil = LogCmdt::logWrite( $caller, $logtext );

and then:

        $app::log->logWrite(( caller(0) )[3], "start testing...");

fin:

        $log->logWrite($config{tool}, "...end");
        $log->logClose();
        $log->logShowError();
$log = new( writeLog, logFile, verboser, logAppend );
nil = $log->logOpen(); creates log file
nil = $log->logClose(); close log file
nil = $log->logWrite( $caller, $logtext ); writes $logtext into logfile
nil = $log->logShowError(); prints all "QS ERROR" lines
